\# 面向对象学生管理系统

Python企业级实战项目 | 求职简历必备 | 全功能完整版



\## 项目介绍

本项目是基于 \*\*Python 面向对象编程（OOP）\*\* 开发的标准化学生信息管理系统，严格遵循 PEP8 代码规范与企业级工程化开发理念。

系统实现了学生信息全生命周期管理（增删改查）、JSON 文件数据持久化、全链路异常安全防护，无第三方依赖、开箱即用，是 Python 后端/AI 应用开发岗位的基础实战项目。



\## 技术栈

\- 核心语言：Python 3.9 \~ 3.12

\- 核心特性：面向对象封装、数据持久化、异常处理、模块化设计

\- 数据存储：本地 JSON 文件

\- 工程规范：Git 版本管理、清晰注释、友好交互



\## 核心功能

学生信息新增：支持录入姓名、年龄、学号唯一标识

学生信息查询：全量列表展示 / 按姓名精准查询

学生信息修改：支持更新学生年龄、学号信息

学生信息删除：支持按姓名删除指定学生

数据持久化：程序重启数据不丢失

异常安全防护：文件异常、输入异常全捕获，程序永不崩溃

控制台菜单交互：简洁易用，操作直观
